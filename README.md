# Image-Classification
This is the project that training a model by using the CNN model to predict the fruit type based on the picture uploaded.

The fruits we have classified are:
1. Apple
2. Banana
3. Guava
4. Lemon
5. Lime
6. Orange
7. Pomegranate

Training Model: Used trained model by ResNet50 to continue training
